How to use Posecode

  1. 1
    Copy the prompt. Click Copy LLM prompt (top right). It teaches any model (ChatGPT, Claude, Gemini) to write the Posecode language.
  2. 2
    Ask an LLM for a movement. Paste the prompt, then ask for what you want: “write a deadlift”, “a hamstring stretch”, “cat-cow”. It replies with a .posecode block.
  3. 3
    Paste it here. Drop the .posecode text into the editor on the left and watch it animate. Out-of-range angles are clamped automatically and flagged below the editor.
